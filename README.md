Dialogic PowerMedia XMS
=======================
Dialogic’s PowerMedia™ XMS is a powerful next-generation software media server that enables standards-based, real-time multimedia communications solutions for mobile and broadband environments. PowerMedia XMS supports standard media control interfaces such as MSML, VXML, NetAnn, and JSR 309, plus a Dialogic HTTP-based version of a RESTful API.


dialogic-jsr309
================
The Dialogic PowerMedia™ XMS JSR309 repository contains demo sample code for J2EE Application Servers using the Dialogic JSR309 connector. The JSR309 specification is a protocol agnostic API for Media Server Control. It provides a portable interface to create media rich applications with IVR, Conferencing, and similar features. 


Repository Contents
===================
**WebRTC_Demo** - J2EE convergent (WebRTC & SIP) demo application divided into two parts. “dlgmsc_conf” part of the WebRTC Conference Demo application is an application which handles business logic and connectivity to Dialogic XMS Media Server via JSR 309 connector. “ConvergedConfereceApp” part of the WebRTC Conference Demo application is an application which handles the Web UI interface. See MS doc files within project for more information on installation and usage.
